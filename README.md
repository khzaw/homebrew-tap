# Khzaw Tap

## How do I install these formulae?

`brew install khzaw/tap/<formula>`

Or `brew tap khzaw/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "khzaw/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
